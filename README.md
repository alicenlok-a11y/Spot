# 🏞️ Spots

A responsive photo gallery project built as part of the TripleTen Software Engineering program (Sprint 3).  
This project demonstrates advanced HTML and CSS layout techniques, including **Flexbox**, **Grid**, and **responsive design** principles.

---

## 🚀 Project Overview

**Spots** is a minimalist photo-sharing webpage that showcases a user profile section and a grid of photo cards.  
It highlights layout management using CSS Grid for flexible image arrangement and Flexbox for component alignment.

### ✨ Features

- Responsive design for various screen sizes
- Semantic HTML structure
- Centered layout with adaptive spacing
- Grid-based image gallery (3-column layout)
- Profile section with edit and add post buttons
- Clean typography using the _Poppins_ font family

---

## 🧱 Tech Stack

- **HTML5** — semantic structure
- **CSS3** — layout and styling
- **Flexbox & Grid** — responsive layout control
- **Normalize.css** — consistent cross-browser rendering
